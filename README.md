# Drako

This is a fork of [Openize.Drako](https://github.com/openize-com/openize-drako-java) intended for use within **jMonkeyEngine**.
If you're looking for a general-purpose, up-to-date Drako library, please use the original repository instead.


### Installation

[Maven Central](https://central.sonatype.com/artifact/org.jmonkeyengine/drako)
